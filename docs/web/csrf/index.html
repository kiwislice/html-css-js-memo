    跨站請求偽造(CSRF)

    跨站請求偽造（英語：Cross-site request forgery），也被稱為 one-click attack 或者 session riding，通常縮寫為 CSRF 或者 XSRF， 是一種挾制用戶在目前已登入的Web應用程式上執行非本意的操作的攻擊方法。跟跨網站指令碼（XSS）相比，XSS 利用的是用戶對指定網站的信任，CSRF 利用的是網站對用戶網頁瀏覽器的信任。

    手法

    假如一家銀行用以執行轉帳操作的URL位址如下：

    http://www.examplebank.com/withdraw?account=AccoutName&amount=1000&for=PayeeName
