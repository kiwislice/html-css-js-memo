    登記Windows服務

    使用工具

    winsw

    流程

    1. 到winsw/releases下載適用的版本(由環境的.NET版本決定)
    2. 從winsw/examples下載xml設定檔
    3. 準備好服務啟動時要執行的目標(EX:.JAR檔)
    4. 將1., 2., 3.放在同資料夾並給相同檔名
    5. 開啟命令提示字元(看狀況可用系統管理員身分執行)，進入該資料夾，執行xxx.exe install即可(xxx自行替換成該exe檔名)

    DEBUG

    如果啟動過程出錯，可至事件檢視器查看
